# QuestionBank

Use this folder to store structured research questions grounded in the local Gaussian Splatting knowledge base.

Good question-bank themes for this repository:

- what kinds of edits preserve geometry best
- how static 3DGS editing methods fail on dynamic or articulated scenes
- what assumptions 4DGS reconstruction methods make about motion and topology
- where text-guided editing breaks view consistency
- what evaluation gaps remain for controllable 4DGS editing
