# paperIDEAs

This directory stores downstream research outputs built on top of the ResearchFlow knowledge base.

Typical contents include idea notes, focused plans, reviewer stress tests, comparison writeups, and implementation-facing research roadmaps.

For the current Gaussian Splatting tracks, especially useful directions include:
- transferring strong 3DGS editing operators into 4DGS settings
- combining reconstruction priors with editing flexibility
- improving temporal consistency without over-constraining edits
- designing evaluation protocols for controllable dynamic-scene Gaussian editing
